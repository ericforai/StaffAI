## 一、了解客户（知彼：摸清对方底细，不做无效沟通）

### 1. 客户基础信息（必查）

| 信息项 | 调研内容 | 记录 |
|--------|----------|------|
| 公司信息 | 全称、所属行业、企业规模、主营业务、近期动态 | |
| 对接人 | 姓名、职位、联系电话、微信、性格沟通风格 | |
| 拜访基础 | 见面地址（精确到楼层会议室）、约定时间、预计时长、拜访事由 | |
| 引荐信息 | 谁引荐的、有无内部对接人、引荐人叮嘱的注意事项 | |

### 2. 客户五要素（必填）

| 要素 | 核心问题 | 自己的记录 |
|------|----------|------------|
| 需求场景 | 客户目前遇到什么问题？核心诉求是什么？ | |
| 使用量 | 预计采购量/使用量是多少？一次性还是长期合作？ | |
| 预算范围 | 有无明确预算？大概区间？谁负责审批？ | |
| 时间节点 | 什么时候想启动？有无明确的时间节点？ | |
| 决策链 | 对接人是决策人吗？还需要对接哪些部门？ | |

### 3. 客户痛点预判（提前写3条以上）

| 痛点类型 | 典型场景 |
|----------|----------|
| 效率痛点 | 流程繁琐、耗时久，想通过合作提升效率 |
| 成本痛点 | 现有合作成本高、性价比低，想降低成本 |
| 风险痛点 | 担心合作风险、产品质量/服务售后问题 |
| 发展痛点 | 想拓展业务、提升竞争力 |

### 4. 调研渠道

公司官网、行业新闻、社交媒体（LinkedIn/微信公众号）、引荐人告知、首次电话/微信沟通提前询问。

## 二、了解产品服务（知己：吃透自身优势，不被问住）

新人最忌讳"自己都不懂产品，还去卖产品"。做到"张口就来"，不看资料也能清晰讲解。

### 1. 产品学习四步法

| 步骤 | 动作 | 目标 |
|------|------|------|
| 定位功能 | 明确核心功能，记住"能解决什么" | 知道功能在哪 |
| 理解作用 | 每个功能对应解决什么痛点 | 能讲清价值 |
| 掌握逻辑 | 了解核心优势与同行的区别 | 不被问住 |
| 串联功能 | 结合客户需求，讲完整解决方案 | 不冗余 |

### 2. 必背内容

- **一句话介绍**：我们公司主要做XX，核心优势是XX，能帮客户解决XX痛点
- **2-3个核心优势**：重点突出，每个配一句简单解释
- **常见问题应答**：提前记3-5个客户常问问题的标准应答

### 3. 同行业案例

准备1-2个"同行业合作案例"简化版：
> "我们之前和XX行业的XX公司合作过，帮他们解决了XX问题，效果很好"

## 三、演示沟通三段论

核心逻辑：**不是推销产品，而是帮客户解决问题**

### 1. 挖坑：提问戳痛点

提前准备3-4个通用问题，根据客户情况灵活提问：

| 问题 | 目的 |
|------|------|
| "您目前在XX环节，有没有遇到比较繁琐、耗时的问题？" | 引出效率痛点 |
| "对于目前的合作/解决方案，您觉得最不满意的地方是什么？" | 挖深层需求 |
| "如果有一款产品/服务能帮您节省成本/提升效率，您最看重哪一点？" | 探决策因素 |
| "您这边对合作的时间节点，有没有明确的要求？" | 确认真实紧迫度 |

### 2. 埋土：给解决方案

根据客户回答，针对性讲解产品/服务：
> "您说目前流程太繁琐，我们的XX功能刚好能解决这个问题，不用手动操作，能节省您大量时间"

### 3. 种花：给额外价值

补充1-2个额外价值，给客户惊喜：

- "除了您关注的这些，我们还能提供免费的售后培训"
- "长期合作的话，我们会有专属对接人，全程跟进"
- "我们还能根据您的需求，定制专属的解决方案"

## 四、报价策略（不慌不乱谈价格）

### 1. 三档报价法（红花绿叶策略）

| 档位 | 定位 | 说明 |
|------|------|------|
| 低档 | 绿叶 | 基础版，满足最低需求 |
| 中档 | 平衡 | 标准版，性价比高 |
| 高档 | 红花 | 完整版，功能齐全，主推 |

**原则：不一开始就报最低价**

### 2. 降价底线

| 规则 | 说明 |
|------|------|
| 降价不超过2次 | 第一次正常沟通，第二次领导特批 |
| 不直接降单价 | 优先用"赠服务、赠时长、赠福利"方式 |
| 不口头承诺 | "我回去帮您申请最优方案，后续给您正式报价单" |

### 3. 必准备资料

- [ ] 报价单（电子版+打印版，标注三档套餐）
- [ ] 套餐对比表（简单明了）
- [ ] 价格谈判话术

## 五、资料设备准备（物理层面不遗漏）

### 1. 纸质资料（必带，精简不冗余）

| 资料 | 说明 |
|------|------|
| 公司简介 | 1-2页精简版，不堆文字 |
| 资质文件复印件 | 营业执照、荣誉等 |
| 产品手册 | 精简版，重点突出核心功能 |
| 成功案例册 | 通用版+1-2个同行业案例 |
| 名片 | 至少10张，双手递出，字朝对方 |
| 报价单打印版 | |
| 空白合作协议模板 | 备用 |
| 笔记本 | 提前写好要问的问题、要记录的要点 |
| 签字笔 | 2-3支，避免没墨 |
| 拜访记录表 | |

### 2. 电子设备（提前检查）

| 设备 | 检查项 |
|------|--------|
| 手机 | 充满电 |
| 电脑/平板 | 充满电、登录好演示账号 |
| 充电宝 | |
| 数据线 | |
| U盘 | 存电子版资料备用 |
| 导航 | 提前定位见面地址 |
| 客户电话 | 存好，避免失联 |

### 3. 个人形象

- 穿着干净整洁，商务休闲/正装（根据行业调整）
- 不带与工作无关的物品
- 细节加分：口罩、纸巾、雨伞、水/口香糖（避免口气）

## 六、话术准备（直接背，不尴尬不冷场）

### 1. 开场话术（30秒内）

> "您好X总/经理，我是XX公司的XX，主要负责XX业务。非常感谢您抽出时间，今天过来主要是想了解下您这边的需求，看看我们的产品/服务能不能帮到您。"

### 2. 被问价格/优势时

> "价格主要看您的具体需求和合作周期，我们有几个不同的套餐，各有优势。我先了解下您的核心需求，再给您推荐最适合、性价比最高的方案，您看可以吗？"

### 3. 不懂/不确定时

> "这个细节我回去跟我们的技术/相关同事确认一下，确保给您准确的答复，不跟您乱承诺，后续第一时间联系您。"

### 4. 客户有异议时

> "我明白您的顾虑，其实很多客户一开始也有这样的担心，不过我们通过XX方式（结合案例/优势），已经帮他们解决了这个问题，您可以放心。"

### 5. 结束话术

> "今天非常感谢您的时间，跟您沟通收获很多。我回去把您的需求整理一下，再把方案和报价完善好，明天发给您，后续我们保持联系，有任何问题您随时找我。"

---

## 七、心态准备（不卑不亢）

### 1. 核心心态

> **1/3必须拿**（明确意向、能推进） + **1/3努力争**（有潜力、需跟进） + **1/3放弃也没关系**（需求不匹配）

不纠结于一次成败。

### 2. 新人避坑提醒

| ❌ 错误做法 | ✅ 正确做法 |
|------------|------------|
| 客户问什么答什么 | 主动提问，引导需求 |
| 乱承诺产品效果 | 实事求是，不夸大 |
| 贬低同行 | 聚焦自身优势，不评价同行 |
| 玩手机/坐姿随意 | 专注沟通，认真记录 |
| 刚见面就催签单 | 循序渐进，先建立信任 |

---

## 八、拜访前最后自检（30秒，全部打勾再出门）

- [ ] 客户五要素、基础信息已全部确认
- [ ] 产品/服务的核心优势、一句话介绍能张口就来
- [ ] 演示/沟通的三段论思路已理顺
- [ ] 报价策略、报价资料已准备好
- [ ] 纸质资料、电子设备、个人物品已全部带齐
- [ ] 核心话术已记牢，知道如何应对常见问题

**补充提示**：提前10分钟到达，在楼下整理好形象；见面时主动微笑、问好，递名片双手递出，字朝对方。
