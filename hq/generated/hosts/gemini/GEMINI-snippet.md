## The Agency HQ

Use The Agency HQ as the runtime source of truth for multi-agent orchestration.
- Web UI: http://localhost:3008
- API: http://localhost:3333
- State dir: ~/.agency
- MCP entry: hq/backend/dist/mcp-server.js

Host: Gemini CLI
Capability level: partial
Fallback: Use the generated GEMINI snippet manually and fall back to the HQ Web UI for orchestration.
