# 团队同步 — 任务页「交付主线」Hero 规则（一段话可直接发群）

**StaffAI 任务工作区（`/tasks`）顶部的「交付主线」模块会只展示一条当前焦点，避免仪表盘信息过载。** 规则是：**当你处在「高级需求向导」时，Hero 永远跟向导进度走（显示第几步、完成条件、回到向导），此时即使审批队列里也有待办，也不会抢焦点——这是刻意设计，避免你在填需求漏斗时被审批打断。** 只有回到**简易模式**时，Hero 才会按 **待处理审批 → 正在执行的任务 → 等待审批的任务 → 其他未完成（按最近更新时间）→ 空闲** 的顺序选焦点；所以如果你同时开着向导和有待批事项，会先看到向导；切回简易创建后，就会看到「前往审批」作为主行动。**这不是 bug**；若业务上需要「审批永远压过向导」，需要单独开需求改 `deriveDeliveryFocus` 的优先级。
